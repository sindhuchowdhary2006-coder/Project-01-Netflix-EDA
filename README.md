# Project 01 - Exploratory Data Analysis (EDA)

## Dataset
Netflix Movies & TV Shows Dataset

## Objective
The goal of this project is to perform Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset. The project involves data cleaning, analysis, visualization, and generating business insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Steps
- Loaded and inspected the dataset
- Cleaned missing values and checked duplicates
- Performed Exploratory Data Analysis
- Created 6 different visualizations
- Generated 5 business insights
- Summarized the findings

## Visualizations
- Bar Chart
- Pie Chart
- Histogram
- Line Chart
- Scatter Plot
- Heatmap

## Business Insights
1. Movies dominate Netflix's content library.
2. The United States has the highest number of titles.
3. Most titles were released between 2016 and 2020.
4. TV-MA is the most common content rating.
5. International Movies are the most common genre.
